# Database Schema (doc-verify-sviit)

This folder contains MySQL schema and seed data.

## Tables
- users (role-based access)
- students (branch, year, photo, contact)
- bus_passes (route, validity, active status)
- documents (encryption key storage)
- verification_results (AI output, match scores)
- audit_log (security trail, IP logging)
- otps (email-based password reset)

## Seed Data
Includes 8 sample students across CSE/IT branches.
